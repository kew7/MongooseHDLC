Project for MS Visual Studio Community 2019

Processing files with the HDLC\PPP\TCP\IP Protocol stack (a file or directory with similar files):
processes the HDLC \ PPP\TCP\IP Protocol stack.... with recording sessions to disk and filtering by TCP ports and IP addresses.
The SOFTWARE input must contain a binary file at the level of the channel bit-oriented HDLC Protocol with the specified stack
protocols'.
The main class is CTCP_IP.
Example of a signal in the Signal folder (HDLC. dat).

Обработчик двоичного потока со стеком протоколов  HDLC\PPP\TCP\IP (файла или каталога с подобными файлами):
     производит обработку стека протоколов HDLC\PPP\TCP\IP.... с  регистрацией соединений и отбора по TCP-портам и IP-адресам.
     На входе ПО должен быть двоичный файл на уровне канального бит-ориентированного протокола HDLC с указанным стеком       
     протоколов.
     Основной класс обработки - CTCP_IP.

Пример сигнала в папке Signal (HDLC.dat).
